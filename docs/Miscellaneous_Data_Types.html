Miscellaneous Data Types

Boolean type

The Boolean Data Type is an unsigned integer that ViviFire interprets as either True or False. Its size may be different on different platforms. If you need a variable with two states — particularly true/false, yes/no, or on/off — declare it as Boolean.

Date type

The Date Data Type is a 64-bit integer that holds both date and time values. Each increment represents 100 nanoseconds starting at midnight of the first of January of the year 1 in the Gregorian calendar. If you need a variable to hold a date, a time, or both, declare it as Date.
