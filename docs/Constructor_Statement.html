Constructor Statement

Initializes the state of a newly created class using the parameters passed to it.

Syntax

Long form
Constructor [parameterList]
    [statements]
End Constructor
Short form
Ctor [parameterList]
    [statements]
End Ctor

Parts

parameterList
Optional
statements
Optional
End Constructor
Terminates the Constructor block.
End Ctor
Terminates the Ctor block.

Remarks

Multiple constructors are permitted within a class if each constructor has a different parameter signature.

All constructors must be grouped together following any state declarations and before any Destructor, Function, Method, or Sub.

The state declaration section at the top of a class can act as a common constructor. Statements placed in this section are executed before any explicit constructor gets called.
