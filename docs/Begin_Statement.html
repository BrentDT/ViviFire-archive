Begin Statement

Creates an object instance that accepts an anonymous array as a parameter.

Syntax

Begin [Shared] className [ [ specializerList ] ] #object [ , parameterList ]
    statements
    ...
End className

Parts

Shared
Optional
Can be placed before or after Begin.
className
Required
specializerList
Optional
#object
Required
parameterList
Optional. One or more expressions separated by commas.
statements
Required one or more object-creation statements (i.e., New, Object, or a nested Begin) or simply a parameter list, which is treated like the parameters to a New statement. Each statement must be placed on a separate line.
End
Terminates the Begin block. The className must be the same as that used in the Begin clause.

Remarks

All objects created by the statements must have a type compatible with the array parameter.

Examples

Listing 1
Begin Menu #File, "&File"
    "&Open...", DoFileOpen
    |----
    "E&xit", DoFileExit
End Menu
