-= Operator

Subtracts the value of an expression from the value of a variable, then assigns the result to the variable.

Syntax

variable -= expression

Parts

variable
Required numeric variable
expression
Required numeric expression

Remarks

Given the above syntax specification is valid code, the following code would be its equivalent.

variable = variable - expression

Like other assignment operators, the -= operator can be used in a property assignment role to call a method that takes a single optional numeric parameter.

Examples

Variable

x = 50
x -= 8
After execution
variablevalue
x42

Property

' Initialize using a normal method call
#person weight(80kg)
' Change weight using method syntax
#person weight(#person weight? - 10kg)
' Change weight using property syntax
#person weight -= 10kg
