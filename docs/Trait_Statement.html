Trait Statement

Defines a trait, a means of describing how a class behaves.

Syntax

Trait name [ Does traitList ]
    [statements]
End Trait

Parts

name
Required identifier
traitList
Optional. One or more traits, seperated by commas.
statements
Optional
End Trait
Terminates the Trait block.
