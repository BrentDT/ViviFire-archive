Trait Statement

Defines a trait, a means of describing how a class behaves.

Syntax

Trait name [ [typeList] ] [ Does traitList ]
    [statements]
End Trait

Parts

name
Required identifier
typeList
Optional. One or more type parameters, each separated by commas, all enclosed by square brackets. See Type List for more information.
traitList
Optional. One or more traits, seperated by commas.
statements
Optional
End Trait
Terminates the Trait block.
