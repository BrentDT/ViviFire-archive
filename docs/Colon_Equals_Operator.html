:= Operator

Assigns a value to a variable.

Syntax

variable := value

Parts

variable
Required variable
value
Required expression of a type compatible with the variable

Remarks

The := operator acts like the = operator to assign a new value to a variable, except that := offers more flexibility in its placement. Unlike the = operator, the := operator's only function is assignment; whereas, the = operator can represent either assignment or comparison, depending on context.

Like other assignment operators, the := operator can be used in a property assignment role to call a method that takes a single parameter.

Examples

Variable

The following code shows how a 3-D application can zero out its x, y, and z variables:

Listing 1
x := y := z := 0
After execution
variablevalue
x0
y0
z0

Property

Listing 2
' Change value using method syntax
#obj value(42)
' Change value using property syntax
