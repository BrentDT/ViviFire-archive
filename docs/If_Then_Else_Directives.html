@If...Then...@Else Directives

Permits the conditional compilation of ViviFire code.

Syntax

@If condition [Then]
    [statements]
[ @ElseIf elseifCondition [Then]
    [statements] ]
...
[ @Else
    [statements] ]
@End If

Parts

condition
Required
statements
Optional
@End If
Terminates the @If statement block.
