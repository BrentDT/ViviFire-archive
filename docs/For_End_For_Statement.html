For...End For Statement

Repeats a block of code for every value in a range.

Syntax

For counter = start To end [ Step step ]
    [statements]
    [afterStatements]
{ End For | Next } [counter]

Parts

counter
Required in the For clause
Optional in the End For and Next clauses
start
Required numeric expression
end
Required numeric expression
step
Optional numeric expression
statements
Optional
afterStatements
Optional statements executed following the repeated statements, or instead of them. For details, see Afterward...Otherwise Clauses.
End For
Terminates the For block, unless Next is used
Next
Terminates the For block, unless End For is used

Remarks

Do not confuse the End For clause with the Exit For statement.
