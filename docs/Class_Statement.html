Class Statement

Declares the name of a class and defines its implementation.

Syntax

[ Abstract ] Class name [ [ genericsList ] ] [ Is superclass ] [ Does traitList ]
    [ Where genericConstraints ]
    [ statements ]
End Class

Parts

Abstract
Optional. Specifies that this is an abstract class, and is ordinarily used as the parent of a concrete class.
name
Required
genericsList
Optional. One or more plain identifiers separated by commas and surrounded by square brackets.
superclass
Optional. The name of another class from which this class inherits methods.
traitList
Optional list of traits, each separated by a comma. For details, see Does Clause (Traits).
genericConstraints
Optional. One or more constraints on a generic parameter separated by commas. See Where Clause (Generics) for details.
statements
Optional
End Class
Terminates the Class block.

Remarks

Examples

Listing 1
Class Person
    Dim myName$, myAge
    Constructor name$, age
        myName$ = name$
        myAge = age
    End Constructor
    Method GetName$
        Return myName$
    End Method
    Method GetAge
        Return myAge
    End Method
End Class
Usage
Require System.IO #out
Person #a, "Alice", 24
Person #b, "Bob", 42
#out #a GetName$? & " is " & #a GetAge? & ""NL""
#out #b GetName$? & " is " & #b GetAge? & ""NL""
End
' Output:
' Alice is 24
' Bob is 42
