Tiny Data Type

Holds signed 8-bit (1-byte) integer values in the range −128 – +127.

Remarks

Default value
When you declare a variable of type Tiny and leave it uninitialized,, its value defaults to zero (0).
Widening
The Tiny data type widens to Short, Integer, Int, Long, Huge, Single, Double, XFP, or Quad without any risk of overflow.
Type characters
Affixing the characters %1 to the end of a variable name declares it as having a type of Tiny.

Examples

Dim foo As Tiny
Dim bar%1
