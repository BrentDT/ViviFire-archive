Rem= Operator

Divides the value of a variable by the value of an expression, then assigns the remainder to the variable.

Syntax

variable Rem= expression

Parts

variable
Required numeric variable.
expression
Required numeric expression

Remarks

The following two statements are functionally equivalent.

variable Rem= expression
variable = variable Rem expression

Because it is derived from the Rem operator, the sign of the variable does not change.

Examples

Variable

Property

' Change value of property
#obj.value Rem= 2
