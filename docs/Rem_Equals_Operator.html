Rem= Operator

Divides the value of a variable by the value of an expression, then assigns the remainder to the variable.

Syntax

variable Rem= expression

Parts

variable
Required numeric variable.
expression
Required numeric expression

Remarks

Given the above syntax specification is valid code, the following code would be its equivalent.

variable = variable Rem expression

Because it is derived from the Rem operator, the sign of the variable does not change.

Like other assignment operators, the Mod= operator can be used in a property assignment role to call a method that takes a single optional numeric parameter.

Examples

Variable

Property

' Change value using method syntax
#obj value(#obj value? Rem 2)
' Change value using property syntax
#obj value Rem= 2
