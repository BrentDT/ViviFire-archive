Date Data Type

Remarks

Default value
Describe
Type characters
Date has no associated type characters.

Date literal

You can specify a Date value using a Date literal. The literal begins with an @ character followed either by a Gregorian date of the form yyyy/mm/dd or a time of the form hh:mm:ss. You can also use either hyphens (-) or dots (.) to separate the parts of a date, if you prefer.

You can specify both a date and time in a single literal by writing both as a single unit, i.e., @yyyy/mm/dd@hh:mm:ss, in that order.

Type conversions

If you convert a Date value to the String type, ViviFire renders the date as the short date format specified by the user's locale and the time either as a 12- or 24-hour format likewise.

Examples

Dim foo As Date
foo = @2013/12/11@10:09:08
