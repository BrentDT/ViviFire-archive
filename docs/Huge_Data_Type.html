Huge Data Type

Holds signed 128-bit (16-byte) integer values in the range −170,141,183,460,469,231,731,687,303,715,884,105,728 – +170,141,183,460,469,231,731,687,303,715,884,105,727.

Remarks

Default value
When you declare a variable of type Huge and leave it uninitialized,, its value defaults to zero (0).
Widening
The Huge data type widens to Single, Double, XFP, or Quad without any risk of overflow.
Type characters
Huge has no associated type characters.

Examples

Dim foo As Huge
