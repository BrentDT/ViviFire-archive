And= Operator

Performs a logical conjunction, then assigns the result to a variable.

Syntax

variable And= expression

Parts

variable
Required numeric variable
expression
Required numeric expression

Remarks

The following two statements are functionally equivalent.

variable And= expression
variable = variable And expression

Examples

Listing 1
x = &HAA
x And= &H3F
After execution
variablevalue
x42
