& Operator

Creates a new string by concatenation.

Syntax

expression1 & expression2

Parts

expression1
Required string expression or an expression that can be converted to a string
expression2
Required string expression or an expression that can be converted to a string

Remarks

If either expression is not of type String, but can be converted to a string, that expression will be converted automatically. If either expression cannot be converted, ViviFire will generate a compile-time error. The #Null object reference is automatically converted to an empty string ("").

Examples

Listing
greeting$ = "Hello," & " world!"
After execution
VariableValue
greeting$"Hello, world!"
