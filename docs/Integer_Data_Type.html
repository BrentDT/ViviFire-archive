Integer Data Type

Holds signed 32-bit (4-byte) integer values in the range −2,147,483,648 – +2,147,483,647.

Remarks

NoteViviFire also has a Int data type. Do not assume that Integer and Int are synonymous.
Default value
When you declare a variable of type Integer and leave it uninitialized,, its value defaults to zero (0).
Widening
The Integer data type widens to Int, Long, Huge, Single, Double, XFP, or Quad without any risk of overflow.
Type characters
Affixing the characters %32 or %S32 to the end of a variable name declares it as having a type of Integer.

Examples

Dim foo As Integer
Dim bar%32
Dim baz%S32
