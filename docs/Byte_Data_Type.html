Byte Data Type

Holds unsigned 8-bit (1-byte) integer values in the range 0–255.

Remarks

Default value
When you declare a variable of type Byte and leave it uninitialized,, its value defaults to zero (0).
Negation
Because Byte is an unsigned type, when you use the unary minus (-) operator on an expression of this type, the value is first converted to a Short then negated.
Widening
The Byte data type widens to Short, UShort, Int, Uint, Integer, UInteger, Long, ULong, Huge, UHuge, Single, Double, XFP, or Quad without any risk of overflow.
Type characters
Byte has no associated type characters.

Examples

Dim foo As Byte
