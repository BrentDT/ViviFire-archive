ULong Data Type

Holds unsigned 64-bit (8-byte) integer values in the range 0–18,446,744,073,709,551,615.

Remarks

ULong is short for "unsigned long integer".

Default value
When you declare a variable of type ULong and leave it uninitialized,, its value defaults to zero (0).
Negation
Because ULong is an unsigned type, when you use the unary minus (-) operator on an expression of this type, the value is first converted to a Huge then negated.
Widening
The ULong data type widens to Huge, UHuge, Single, Double, XFP, or Quad without any risk of overflow.
Type characters
ULong has no associated type characters.

Examples

Dim foo As ULong
