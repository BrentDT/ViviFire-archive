Function Statement

Defines a procedure that returns a value.

Syntax

[Shared] Function name [typeList] [parameterList]
    [optionalParams] [ Where genericConstraints ]
    [statements]
End Function

Parts

Shared
Optional; Specifies that the function is associated with its surrounding object and not an instance of that object.
name
Required
typeList
Optional. See Type List for details.
parameterList
Optional. Zero or more local variables, separated by commas, whose values are assigned when the function is called. See Parameter List for more information.
returnType
Optional 
As dataType
or
In unit
dataType

unit
Describe
optionalParams
Optional. See Parameter List for more information.
genericConstraints
Optional. See Where Clause (Generics) for details.
statements
Optional
End Function
Terminates the Function block.

Remarks

Do not confuse the End Function clause with the Exit Function statement.
