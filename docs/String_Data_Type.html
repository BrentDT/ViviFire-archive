String Data Type

Holds sequences of 16-bit (2-byte) Unicode character code points. A string can contain from zero to ~ characters.

Remarks

Default value
When you declare a variable of type String and leave it uninitialized,, its value defaults to the empty string ("").
Type characters
Affixing the $ character to the end of a variable name declares it as having a type of String.
