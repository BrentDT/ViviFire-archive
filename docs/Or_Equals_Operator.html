Or= Operator

Performs an inclusive logical disjunction, then assigns the result to a variable.

Syntax

variable Or= expression

Parts

variable
Required numeric variable
expression
Required numeric expression

Remarks

The following two statements are functionally equivalent.

variable Or= expression
variable = variable Or expression

Examples

Listing 1
x = &H22
x Or= &H8
After execution
variablevalue
x42
