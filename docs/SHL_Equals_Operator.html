SHL= Operator

Performs a arithmetic left shift on the value of a variable and assigns the result to the same variable.

Syntax

variable SHL= amount

Parts

variable
Required variable of an integral type.
amount
Required integer expression specifying the number of bits to shift.

Remarks

The following two statements are functionally equivalent.

variable SHL= amount
variable = variable SHL amount
