Decision Structures

ViviFire allows you to test for various conditions and perform different actions depending on the result. You can test for true or false conditions, various values of an expression, or when code cannot continue under exceptional conditions.

If...Then...Else construction

The If...Then...Else construction allows you to test one or more conditions, then execute one or more statements depending on each condition. The following list summarizes the various ways it can be used:

  • Execute some statement(s) if a condition is true.
  • Execute some statement(s) if a condition is false.
  • Execute some statement(s) if a condition is true, or some other(s) if it is false.
  • Test another condition if a preceding condition is false.

See If...Then...Else Statement for more information.

Select...Case construction

The Select...Case construction allows you to test an expression for one of various values, then execute one or more statements if a match is or is not found.

See Select...Case Statement for more information.

Try...Catch...Finally construction

The Try...Catch...Finally construction allows you to execute code in an environment such that exceptional conditions can be caught in an orderly manner. One or more statements can execute depending on the type of exception. There is also a way to specify one or statements that must always execute whether an exception has occurred or not.

See Try...Catch...Finally Statement for more information.
