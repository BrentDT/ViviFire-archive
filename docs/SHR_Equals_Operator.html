SHR= Operator

Performs a arithmetic right shift on the value of a variable and assigns the result to the same variable.

Syntax

variable SHR= amount

Parts

variable
Required numeric variable.
amount
Required integer expression specifying the number of bits to shift.

Remarks

Given the above syntax specification is valid code, the following code would be its equivalent.

variable = variable SHR amount

Like other assignment operators, the SHR= operator can be used in a property assignment role to call a method that takes a single optional numeric parameter.

Examples

Variable

Property

Listing 2
' Change value using method syntax

' Change value using property syntax
