+= Operator

Adds or concatenates the value of an expression to the value of a variable, then assigns the result to the variable.

Syntax

variable += expression

Parts

variable
Required numeric or string variable
expression
Required expression of a type compatible with variable

Remarks

Given the above syntax specification is valid code, the following code would be its equivalent.

variable = variable + expression

Like other assignment operators, the += operator can be used in a property assignment role to call a method that takes a single optional numeric or string parameter.

Examples

Variable

Listing 1
x = 30
x += 12
After execution
variablevalue
x42

Property

Listing 2
' Change value using method syntax
#obj value(#obj value? + 1)
' Change value using property syntax
#obj value += 1
