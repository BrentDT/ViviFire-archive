Long Data Type

Holds signed 64-bit (8-byte) integer values in the range −9,223,372,036,854,775,808 – +9,223,372,036,854,775,807.

Remarks

Default value
When you declare a variable of type Long and leave it uninitialized,, its value defaults to zero (0).
Widening
The Long data type widens to Huge, Single, Double, XFP, or Quad without any risk of overflow.
Type characters
Affixing the characters %8 to the end of a variable name declares it as having a type of Long.

Examples

Dim foo As Long
Dim bar%8
