Abstract

Specifies that a declared element is incomplete or lacks implementation.

Remarks

Describe

Rules

Instantiation
You cannot create an instance of an abstract class.
Combined modifiers
You cannot combine Abstract with other modifiers such as Shared and Override in the same declaration.
Subclassing
When you derive a non-abstract class from an abstract class, the new class must provide implementations of all abstract methods that it inherited.
Declaration contexts
You can declare abstract methods only within an abstract class.

Applies to
