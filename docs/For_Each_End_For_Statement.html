For Each...End For Statement

Repeats a block of code for every element in a data structure.

Syntax

For Each element In group
    [statements]
    [afterStatements]
{ End For | Next }

Parts

element
Required
group
Required
statements
Optional
afterStatements
Optional statements executed following the repeated statements, or instead of them. For details, see Afterward...Otherwise Clauses.
End For
Terminates the For Each block, unless Next is used
Next
Terminates the For Each block, unless End For is used

Remarks

Do not confuse the End For clause with the Exit For statement.
