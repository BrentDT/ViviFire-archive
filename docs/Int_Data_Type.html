Int Data Type

Holds signed integer values guarenteed to be at least 32 bits (4 bytes) and provides the best performance. On 32-bit architectures, values are in the range −2,147,483,648 – +2,147,483,647.

Remarks

Int is short for "integer".

NoteViviFire also has a Integer data type. Do not assume that Int and Integer are synonymous.
Default value
When you declare a variable of type Int and leave it uninitialized,, its value defaults to zero (0).
Widening
The Int data type widens to Long, Huge, Single, Double, XFP, or Quad without any risk of overflow.
Type characters
Affixing the % character to the end of an variable name declares it as having a type of Int.

Examples

Dim foo As Int
Dim bar%
