Short Data Type

Holds signed 16-bit (2-byte) integer values in the range −32,768 – +32,767.

Remarks

Default value
When you declare a variable of type Short and leave it uninitialized,, its value defaults to zero (0).
Widening
The Short data type widens to Integer, Int, Long, Huge, Single, Double, XFP, or Quad without any risk of overflow.
Type characters
Affixing the characters %16 or %S16 to the end of a variable name declares it as having a type of Short.

Examples

Dim foo As Short
Dim bar%16
Dim baz%S16
