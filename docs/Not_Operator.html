Not Operator

Performs a logical negation.

Syntax

Not expression

Parts

expression
Required numeric expression

Remarks

The Not operator inverts the bits of the given expression. Every bit that was set to one becomes a zero, and every bit that was zero becomes a one.

Examples

Listing 1
pattern = &B101010
inverse = Not pattern
