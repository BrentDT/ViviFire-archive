Typographic Conventions

ExamplesDescription
 
Class, For, Loop, Trait
Language keywords have initial capital letters and are formatted as shown in the example.
condition, #object
Placeholders for information that you supply are formatted as shown in the example.
[ Case ], [ expressionList ]
In syntax, optional items are enclosed in square brackets.
[ [ specializerList ] ]
In syntax, square brackets nested within brackets are syntactically significant.
{ While | Until }
In syntax, when you must make a choice between two or more items, the items are enclosed in curly braces and separated by vertical bars. You must select one, and only one, of the items.
[ + | * ]
In syntax, when you have the option of selecting between two or more items, the items are enclosed in square brackets and separated by vertical bars. You can select any combination of the items, or no item.
[ , parameter ] ...
In syntax, an ellipsis indicates that the preceding element or group can be repeated as many times as needed.
' ...
In listings, a comment with ellipsis indicates that code has been omitted to make the example code clearer.
