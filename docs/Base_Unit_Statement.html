Base Unit Statement

Defines a physical quantity and its standard measure.

Syntax

Base Unit name In unit [ = expression ]

Parts

name
Required. A physical quantity, such as length or time.
unit
Required. The standard unit for measuring the given physical quantity.
expression
Optional. Relates a derived physical quantity to its parents.

Remarks

Describe

Examples

Listing 1: Simple units
Base Unit distance In meters
Base Unit mass In kilograms
Base Unit time In seconds
Listing 2: Derived units
Base Unit velocity In meters/second = distance/time
