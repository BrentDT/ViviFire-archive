If...Then...Else Statement

Describe

Syntax

Multiple-line form
If condition [Then]
    [statements]
[ ElseIf elseifCondition [Then]
    [elseifStatements] ]
...
[ Else
    [elseStatements] ]
End If
Single-line form
If condition Then statement [ Else elseStatement ]

Parts

condition
Required expression
Then
Optional in the multiple-line form
Required in the single-line form
statement(s)
Optional in the multiple-line form
Required in the single-line form
elseifCondition
Required if ElseIf is present
elseifStatements
Optional
elseStatement(s)
Optional
End If
Terminates the multiple-line form
