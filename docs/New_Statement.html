New Statement

Creates an object instance and initializes its state.

Syntax

Normal form
[ Shared ] [ New ] className [ [ specializerList ] ] #object [ , parameterList ]
Subclass constructor form
New [ [ specializerList ] ] #Base [ , parameterList ]

Parts

New
Optional in the normal form
Required in the subclass constructor form
Shared
Optional
className
Required in the normal form. It is omitted by the subclass constructor form where it is implicit.
specializerList
Required only if className is a generic type. One or more data types separated by commas and enclose in square brackets.
#object
Required in the normal form. It is substituted with #Base in the subclass constructor form.
parameterList
Optional. One or more expressions separated by commas.

Remarks

Examples

Listing 1
' Find all text files in CWD
Files #fs, "*.txt"
