Enum Statement

Declares an enumeration and defines its members.

Syntax

Enum [ enumerationName ] [ Step [ + | * ] step ]
    memberList
End Enum

Parts

enumerationName
Optional identifier for the enumeration
step
Optional; Specifies the default amount by which implicit members increase. Optionally, a plus (+) or asterisk (*) may precede the number, which specifies increasing values by addition (the default) or multiplication, respectively.
memberList
Required list of member specifications, one per line.
Syntax
memberName [ = initializer ]
memberName
Required identifier
initializer
Optional constant integer expression
End Enum
Terminates the Enum block.

Remarks

An enumeration is a tightly coupled collection of numeric constants. They can make code easier to read and modify by not relying on arbitrary literals scattered throughout the code. In most cases, there is no need to explicitly set a member's value. The language can guarantee that each member is unique.

The step amount must be a natural number (1, 2, 3, etc.). If no step is specified, it defaults to "+1".

If the first member is not given a value explicitly, its value defaults to 1. Members of an enumeration can be positive or negative. Members can have the same value; however, the names within an enumeration must be unique.

Examples

Listing 1
Enum fruit
    apple
    banana
    cherry
End Enum
Listing 2
Enum days Step * 2
    Mon
    Tue
    Wed
    Thu
    Fri
    Sat
    Sun
End Enum
weekend = Sat Or Sun
