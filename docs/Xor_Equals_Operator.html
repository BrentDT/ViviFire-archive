Xor= Operator

Performs a logical exclusion, then assigns the result to a variable.

Syntax

variable Xor= expression

Parts

variable
Required numeric variable
expression
Required numeric expression

Remarks

The following two statements are functionally equivalent.

variable Xor= expression
variable = variable Xor expression

Examples

Listing 1
x = &H7F
x Xor= &H55
After execution
variablevalue
x42
