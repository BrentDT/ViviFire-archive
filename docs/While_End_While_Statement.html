While...End While Statement

Repeatedly executes a block of code while a condition is true.

Syntax

While condition
    [ statements ]
    [ afterStatements ]
{ End While | Wend }

Parts

condition
Required Boolean expression
statements
Optional
afterStatements
Optional statements executed following the repeated statements, or instead of them. For details, see Afterward...Otherwise Clauses.
End While
Terminates the While block, unless Wend is used
Wend
Terminates the While block, unless End While is used

Remarks

Do not confuse the End While clause with the Exit While statement.
