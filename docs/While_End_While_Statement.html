While...End While Statement

Repeatedly executes a block of code while a condition is true.

Syntax

While condition
    [statements]
    [afterStatements]
{ End While | Wend }

Parts

condition
Required Boolean expression
statements
Optional
afterStatements
Optional statements executed following the repeated statements, or instead of them. For details, see Afterward...Otherwise Clauses.
End While
Required to terminate the While block, unless Wend is used
Wend
Required to terminate the While block, unless End While is used

Remarks

Do not confuse the End While clause with the Exit While statement.

Use the While...End While structure when you need to repeat a block of statements an indefinite number of times, so long as a condition remains true. Other looping structures can offer more flexibility or greater efficiency under certain circumstances. See the See also section below for details.

Normal loop execution

If condition is true, the statements are executed. From the End While clause, control returns to the While clause, where the condition is tested again. If it remains true, the process repeats. If it becomes false, control passes to the statement following End While.

The While clause always tests the condition before it starts the loop. If the condition is initially false, the loop never executes at all.

Exit While

The Exit While statement provides another way to exit a While loop. Exit While immediately transfers control to the statement following End While.
