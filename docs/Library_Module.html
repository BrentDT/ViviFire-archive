Library Module

Defines a source file containing shared code that can be used by other modules.

Syntax

Library name [ , attribute ]...
[ Where attribute ]
...
[statements]

Parts

name
Required. A plain identifier.
attribute
Optional. One or more name-value pairs that can be listed in the Library clause separated by commas, or immediately following the clause, one per line, following the Where keyword.
Syntax
attribName = attribValue
attribName
Required identifier
attribValue
Required. A literal value; either a string literal or version literal.
statements
Optional

Remarks

The Library statement must be the first non-commented line of code in the source file of a library module. A source module lacking this statement thus defaults to a user module, whose purpose is to define the entry point of a program.
