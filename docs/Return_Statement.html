Return Statement

Describe

Syntax

Return [expression]

Parts

expression
Optional

Remarks

A Return statement without an expression is equivalent to an Exit Function, Exit Method, or Exit Sub statement in its respective context. It causes execution to exit out of the procedure in which it appears and return to the statement that invoked the procedure.
