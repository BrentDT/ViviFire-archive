Destructor Statement

Defines the code to execute when an instance of a class is destroyed.

Syntax

Long form
Destructor
    [statements]
End Destructor
Short form
Dtor
    [statements]
End Dtor

Parts

statements
Optional
End Destructor
Terminates the Destructor block.
End Dtor
Terminates the Dtor block.

Remarks

Typically, a destructor is needed to free a system resource, such as an Internet socket, for example.

Only a single destructor is permitted within a class. The destructor must be placed after any and every Constructor and before any Function, Method, or Sub.
