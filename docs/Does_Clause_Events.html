Does Clause (Events)

Specifies that a Sub procedure can handle the specified event.

Syntax

Does eventName

Parts

eventName
Required. The name of an event as defined by an Event statement.

Remarks

The Does clause does two things for a Sub procedure. First, it modifies the procedure so it becomes an event handler. Secondly, it injects the parameters defined by the Event statement into the procedure where they become available, as if they were defined locally.
