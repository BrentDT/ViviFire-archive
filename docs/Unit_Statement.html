Unit Statement

Defines a unit of measure.

Syntax

Unit [ [parameter In unit] ] [ (SI) ] name(s) [ = expression ]

Parts

parameter
Optional. A numeric variable that can be used in the expression, acting as a placeholder when defining dimensionless physical quantities, such as temperature.
unit
A related unit to the one being defined
SI
Optional indication of an SI unit. You can use any plain identifier enclosed in parentheses, since it is used as a flag and is otherwise ignored. However, you are strongly encouraged to standardize on "SI" for the sake of consistency.
name(s)
Required. One or more names separated by spaces. All names are synonymous. Plural forms can be created by using parentheses.
expression
Optional. A conversion factor.
