Where Clause (Generics)

Specifies constraints placed on generic parameters.

Syntax

Where constraint [ , constraint ]...

Parts

constraint
Required. One or more constraints to place on a generic parameter, each separated by a comma.
Syntax
[ lowerBound { < | <= } ] genericName { < | <= } upperBound [ Does boundingTrait ]
or
genericName Does boundingTrait
genericName
One of the previously supplied generic parameters.
upperBound
The most basic class from which genericName can inherit.
lowerBound
The most specialized class from which genericName can inherit.
boundingTrait
A trait that genericName must implement.
