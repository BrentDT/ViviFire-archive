Rem Operator

Divides two numbers and returns the remainder.

Syntax

dividend Rem divisor

Parts

dividend
Required numeric expression
divisor
Required numeric expression

Remarks

The result has the same sign as the dividend. The similar Mod operator differs by taking the sign of the divisor instead. One way to remember this is that there is an "e" in both "Rem" and "dividend".
