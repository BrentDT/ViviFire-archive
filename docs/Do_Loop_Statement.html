Do...Loop Statement

Repeats a block of code while a condition is true or until a condition becomes true.

Syntax

Preconditional form
Do { While | Until } condition
    [ statements ]
    [ afterStatements ]
Loop
Postconditional form
Do
    [ statements ]
    [ afterStatements ]
Loop { While | Until } condition

Parts

Do
Required
While
Required unless Until is used
Until
Required unless While is used
condition
Required Boolean expression
statements
Optional
afterStatements
Optional statements executed following the repeated statements, or instead of them. The post-conditional form places constraints on these statements. For details, see Afterward...Otherwise Clauses.
Loop
Required
