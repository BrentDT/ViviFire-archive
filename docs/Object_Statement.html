Object Statement

Defines a singleton object.

Syntax

[Shared] Object #handle [ Is superclass ] [ Does traitList ]
    [statements]
End Object

Parts

Shared
Optional
#handle
Required
superclass
Optional
traitList
Optional list of traits, each separated by a comma. For details, see Does Clause (Traits).
statements
Optional
End Object
Terminates the Object block.
